# 🔍 Visualizador de Perfil do GitHub

Uma aplicação simples e intuitiva que permite aos usuários buscar e visualizar perfis do GitHub, exibindo informações básicas e os repositórios mais recentes.

## ✨ Funcionalidades

- 🔎 **Busca por Usuário:** Basta digitar o nome de usuário do GitHub para encontrar o perfil.
- ⌨️ **Atalho de Teclado:** Pressione a tecla **Enter** para disparar a busca rapidamente.
- 📂 **Repositórios:** Exibe os repositórios do usuário, organizados para fácil visualização.
- 📱 **Design Responsivo:** Adaptado para diferentes tamanhos de tela.
- 🔄 **Feedback Visual:** Indica quando os dados estão sendo carregados.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica da aplicação.
- **CSS3:** Estilização com animações e layout responsivo.
- **JavaScript (ES6+):** Lógica da aplicação, manipulação do DOM e consumo de API.
- **GitHub API:** Fonte de dados para perfis e repositórios.
- **DevIcons:** Ícones para melhorar a interface visual.

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/ramonvieira41/visualizador-perfil-github.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd visualizador-perfil-github
   ```
3. Abra o arquivo `index.html` em seu navegador.
